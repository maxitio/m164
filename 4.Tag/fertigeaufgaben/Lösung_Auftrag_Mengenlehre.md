## Aufgabe 1

Gegebene Mengen:
A={c,e,z,r,d,g,u,x}, B={c,e,g}, C={r,d,g,t}, D={e,z,u}, E={z,r,u}

**a) B ⊂ A**
**Wahr**. Alle Elemente von B (c, e, g) sind vollständig auch in A enthalten.

**b) C ⊂ A**
**Falsch**. C enthält das Element `t`, welches nicht in A vorkommt.

**c) E ⊂ A**
**Wahr**. Alle Elemente von E (z, r, u) sind vollständig auch in A enthalten.

**d) B ⊂ C**
**Falsch**. Die Elemente `c` und `e` aus B sind nicht in C enthalten.

**e) E ⊂ C**
**Falsch**. Die Elemente `z` und `u` aus E sind nicht in C enthalten.

## Aufgabe 2

Gegebene Mengen:
A={1;2;3;4;5}, B={2;5}, C={3;5;7;9}

**a) Bestimmen Sie A ∩ B** (Schnittmenge)
A ∩ B = {2; 5}

**b) Bestimmen Sie A ∪ C** (Vereinigungsmenge)
A ∪ C = {1; 2; 3; 4; 5; 7; 9}

**c) Bestimmen Sie B^c** (Komplementärmenge der Menge B)
*Anmerkung:* Komplement ist auf eine Grundmenge bezogen. Nehmen wir an, die Grundmenge ist die Vereinigungsmenge aller auftretenden Zahlen G = A ∪ B ∪ C = {1; 2; 3; 4; 5; 7; 9}. 
B^c = {1; 3; 4; 7; 9}

**d) Bestimmen Sie B \ C** (Differenzmenge: Was ist in B, aber nicht in C)
B \ C = {2}

**e) Bestimmen Sie C \ B** (Differenzmenge: Was ist in C, aber nicht in B)
C \ B = {3; 7; 9}

## Aufgabe 3

1. **⊂ (Teilmenge)**
   Wir haben eine Menge `Gaming` von Spielgeräten und eine Menge `Konsolen`.
   Gaming = {PC, PlayStation, Xbox, Switch}
   Konsolen = {PlayStation, Xbox, Switch}
   *Die Konsolen sind eine Teilmenge der Gaming-Geräte:* `Konsolen ⊂ Gaming`

2. **∩ (Schnittmenge)**
   Personen, die Pizza mögen = {Anna, Max, Tom}
   Personen, die Burger mögen = {Max, Tom, Sarah}
   *Die Schnittmenge (Personen, die beides mögen) ist:* `{Max, Tom}`

3. **∪ (Vereinigungsmenge)**
   Filme angesehen von Person 1 = {Inception, Matrix}
   Filme angesehen von Person 2 = {Matrix, Avatar}
   *Alle insgesamt gesehenen Filme (Vereinigung):* `{Inception, Matrix, Avatar}`

4. **\ (Differenzmenge)**
   Gekaufte Zutaten = {Mehl, Zucker, Eier, Milch}
   Bereits verbrauchte Zutaten = {Mehl, Milch}
   *Die restlichen verfügbaren Zutaten (Gekaufte Zutaten \ Verbrauchte Zutaten):* `{Zucker, Eier}`
